# YouTube Shorts Blocker

A sleek, modern browser extension designed to reclaim your YouTube experience by blocking Shorts across the platform. Built with Manifest V3 for superior performance and security.

## Features
- **Granular Control**: Toggle blocking for different parts of YouTube independently.
- **Sidebar Hiding**: Removes the "Shorts" tab from the side navigation menu.
- **Main Grid Blocking**: Hides Shorts sections from the home feed and search results.
- **Automatic Redirection**: Automatically opens Shorts as standard YouTube videos.
- **Modern UI**: A premium glassmorphism-style popup with a clean toggle interface.
- **Privacy Focused**: No tracking, no data collection, and minimal resource usage.

## Installation

### Temporary Installation (For testing)
1. Open Firefox and go to `about:debugging`.
2. Click on **"This Firefox"**.
3. Click **"Load Temporary Add-on"**.
4. Select the `manifest.json` file from this directory.
